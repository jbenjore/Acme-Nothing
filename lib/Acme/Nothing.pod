=head1 NAME

Acme::Nothing - No more module loading !

=head1 SYNOPSIS

Stops your script from loading any more modules.

    use Acme::Nothing;
    use Improbable; # Nope!
    use Fish;       # Not this either!
    use CGI;        # Still not loading anything
